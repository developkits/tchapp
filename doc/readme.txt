概述

使用Web做UI，csharp编程，构建跨平台桌面软件，项目目标是打造一个框架，做完相关的基础设施，让使用者能够只关注业务开发，而不需要重新构建基础设施。
项目应该包括：窗口管理API，跨语言（JS，C++，C#）交互API
项目应设计为可模块重组替换，平台无关，可运行于linux和windows，OSX，语言分离，c++集成可选，dotnet core集成可选，asp.net core 集成可选，mvc集成可选，可集成node.js，可任意组合，可扩展

技术选择：
采用cef3项目渲染Web，管理窗口，cef3本身可跨平台
采用.net core做csharp基础，.net core本身可跨平台

相似项目：Electron
